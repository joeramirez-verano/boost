# Button

- Most versatile component of the project. Offer variety of customization.
- We can use it as react component.
- Also it called by gutenberg button block.
- Each brand have 4 different versions (See type props) of it.
- As well as 4 different sizes.

## Design

- [Design is here](https://www.figma.com/file/AqGeBrT86Lc4PIZzcPw9R8/Brand-UI-Working-Project?type=design&node-id=22%3A1815&mode=design&t=m2Bbd5R44QvjsaVj-1)

## Functions

- Powerfull props to customization to next lavel.
- Can be used as button or link.
- `onClick` function can be passed.
- Can be used as icon only button.
