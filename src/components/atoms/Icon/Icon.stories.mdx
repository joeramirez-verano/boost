# Icon

- Use this component to display a Custom icon.
- All icons stored in `icons` object in `icons.js` file.
- Each items in `icons` object is a key-value pair where the icon name is a arrow function as a key and value return jsx of the svg. So, that it formate clearly in editor.

## Functions

- The `name` prop is required and should be a string.
- The `size` prop is optional and should be a string.
- The `size` prop can be one of `sm`, `md`, or `lg` and defaults to `md`.
